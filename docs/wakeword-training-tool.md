# 唤醒词训练工具使用指南

## 概述

唤醒词训练工具允许你创建自定义唤醒词或个性化现有唤醒词，让 TryVoice 更好地识别你的声音。

## 访问训练工具

使用前端 API 启动训练工具：

```typescript
import { startTrainingTool } from './wakeword/training-tool';

// 启动训练工具
startTrainingTool();
```

## 训练流程

### 第一步：输入唤醒词

- 输入你想要的唤醒词文本（如「小助手」）
- 建议 2-8 个字
- 避免使用常见词或日常用语
- 推荐使用独特的词组

### 第二步：采集样本

**麦克风录制（推荐）**
1. 点击「开始录音」按钮
2. 对着麦克风清晰地说出唤醒词
3. 点击「停止录音」或等待自动停止
4. 重复录制至少 5 个样本（推荐 10-20 个）

**TTS 补充**
- 如果样本数量不足，可以点击「用 TTS 补充」
- 系统将自动生成语音样本

### 第三步：训练模型

- 点击「下一步：开始训练」
- 等待训练完成（通常需要 1-3 分钟）
- 可以随时点击「取消训练」

### 第四步：验证模型

**批量回测**
- 点击「运行批量回测」
- 系统将使用已录制的样本测试识别率
- 显示成功率统计

### 第五步：安装模型

**立即安装**
- 点击「立即安装」
- 模型将热加载到当前实例
- 立即可用于唤醒词检测

**下载模型**
- 点击「下载模型文件」
- 获取 .onnx 文件用于备份或部署

## API 端点

| 端点 | 方法 | 用途 |
|------|------|------|
| `/wakeword/train/samples` | POST | 上传麦克风录制样本 |
| `/wakeword/train/tts-generate` | POST | 生成 TTS 补充样本 |
| `/wakeword/train/start` | POST | 启动训练任务 |
| `/wakeword/train/status/{taskId}` | GET | 查询训练进度 |
| `/wakeword/train/result/{taskId}` | GET | 获取训练结果 |
| `/wakeword/train/{taskId}` | DELETE | 取消训练任务 |
| `/wakeword/train/install` | POST | 安装模型到项目 |

## 注意事项

- 训练新唤醒词需要后端服务器支持
- 个性化微调在浏览器端完成，不需要上传数据
- 如果成功率低于 80%，建议返回补充样本重新训练

## 常见问题

**Q: 训练需要多长时间？**
A: 新建唤醒词约 2-5 分钟，个性化微调约 30 秒。

**Q: 可以训练多少个唤醒词？**
A: 没有硬性限制，但建议不超过 5 个以避免冲突。

**Q: 为什么成功率很低？**
A: 可能是样本质量不足。尝试在安静环境下录制，确保发音清晰。
